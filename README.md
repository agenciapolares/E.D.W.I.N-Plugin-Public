# E.D.W.I.N-Plugin-Public
O Edwin Plugin é uma ferramenta poderosa para o KiCad que automatiza o posicionamento e a rotação de matrizes de LEDs a partir de layouts gerados no sistema web interativo do Edwin.

Como baixar e instalar no KiCad

Baixar o Plugin do GitHub Acesse a página do repositório no GitHub. Navegue até a seção [Releases] (Lançamentos) no lado direito da tela. Faça o download do arquivo compactado mais recente: edwin_plugin.zip.
Instalar no KiCad Extraia o conteúdo do arquivo edwin_plugin.zip que você acabou de baixar. Copie a pasta resultante da extração (certifique-se de que é a pasta que contém o plugin). Cole a pasta copiada dentro do seguinte diretório de plugins do KiCad no seu computador: C:\Program Files\KiCad\10.0\share\kicad\scripting\plugins (Dica: Se a pasta plugins não existir, você deve criá-la). Feche o KiCad caso ele esteja aberto. Abra o KiCad novamente para carregar o plugin. Ele estará disponível no Editor de PCB. Como usar
O fluxo de trabalho funciona em duas etapas: primeiro você projeta no sistema web, e depois importa os dados prontos para o seu projeto no KiCad.

Passo 1: Exportando o .edwin do Sistema Web Acesse a plataforma web do Edwin: https://edwinweb.vercel.app/ Utilize as ferramentas visuais para gerar sua matriz de LEDs (defina colunas, linhas, layouts circulares, espaçamentos e rotações). Após finalizar o design, clique no botão Exportar Layout (KiCad). Seu navegador fará o download de um arquivo com a extensão .edwin. Esse arquivo é criptografado e contém as coordenadas e propriedades exatas do seu layout. Passo 2: Importando no KiCad Abra o seu projeto no KiCad e entre no Editor de Placas de Circuito Impresso (PCB Editor). Certifique-se de que os LEDs já foram inseridos na placa (via Atualizar PCB a partir do Esquemático). A referência geralmente é D1, D2, etc. Observe a barra superior de ferramentas do KiCad. Clique no Ícone do Edwin Plugin. Uma pequena interface será aberta. Clique no botão de buscar/abrir arquivo. Selecione o arquivo .edwin que você baixou anteriormente do sistema web e clique em Abrir/Importar. Pronto! O Edwin irá organizar, posicionar e aplicar as rotações adequadas em todos os LEDs na placa de forma totalmente automática. Dica Importante Para que o plugin funcione corretamente, os componentes no KiCad devem estar nomeados e enumerados de forma sequencial, compatível com o layout gerado. Caso ocorra algum erro, verifique se está utilizando o arquivo .edwin mais atualizado e se não modificou manualmente a extensão do arquivo.
